# Colour-Bump
Try to complete the tasks by matching the colored cups

# Game Description
In total, we have 10 different cups and 10 different tasks. We earn points from tasks using 3 life bars and a score bar.

Our goal is to complete the cup matching game in an endless sequence without reducing the number of moves (20) and lives (3).

Matches;

Glass1  X Glass1-->Glass2

Glass2  X Glass2-->Glass3

Glass3  X Glass3-->Glass4

Glass4  X Glass4-->Glass5

Glass5  X Glass5-->Glass6

Glass6  X Glass6-->Glass7

Glass7  X Glass7-->Glass8

Glass8  X Glass8-->Glass9

Glass9  X Glass9-->Glass10

Glass10  X Glass10-->Destroy
